# SQL Sales Data Cleaning 
SQL data cleaning project using MySQL

This project demonstrates how to clean raw sales data using MySQL.

## Cleaning Steps
- Created a working copy of raw dataset
- Removed duplicate records
- Handled missing values
- Standardized text columns
- Converted inconsistent date formats
- Created TransactionType column
- Cleaned email domains

- MySQL
